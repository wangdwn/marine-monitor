## [v1.1.0] 2025-01-10 — 规划局三大优势融合

### 新增模块
- **资源调查监测页面** (`/survey`)
  - 海域面积、海岛、海岸线、地质、湿地6大核心指标
  - 9大调查类别分布
  - 8个监测站点网络
  
- **规划用地保障页面** (`/planning`)
  - 空间规划合规率、用海审批效率、项目保障数4大KPI
  - 6大规划类型完成率进度追踪
  - 三区三线、功能区划等专项指标

- **安全屏障态势页面** (`/safety`)
  - 地质灾害、风暴潮、海岸侵蚀、生态修复6大统计
  - 4级风险分布可视化
  - 实时安全监测预警列表

### 数据看板升级
- 核心KPI调整为规划局视角（海域确权、海岸线保护率、地灾隐患点）

### 数据库扩展
- 新增4张核心表：survey_data、planning_data、safety_data、monitoring_stations
- 新增3个tRPC router：survey、planning、safety

## [v1.0.0] 2025-01-08 — 基础平台上线

### 核心功能
- 数据看板（Dashboard）— 4大KPI、趋势图、赛道分布
- 产业地图（Map）— 资源点位可视化、图层筛选
- 分析报告中心（Reports）— 列表/详情、分页筛选
- 政策解读（Policies）— 层级筛选、AI摘要
- 用户中心（Profile）— 资料管理
- 管理后台（Admin）— 统计看板、用户/报告管理

### 技术架构
- React 19 + TypeScript + Tailwind CSS + shadcn/ui
- tRPC + Drizzle ORM + Hono + MySQL
- Kimi OAuth 2.0 认证体系
